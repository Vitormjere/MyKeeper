# 💻 MyKeeper

> Conheça a mais nova chave para o sucesso no gerenciamento do seu estoque alimentício: o MyKeeper! Projeto desenvolvido por Lucas Vianna, Rodrigo Yuji, Alan Mendes, José Ricardo e Vitor Miranda.  

O sistema consiste em um gerenciador de estoque alimentício, desenvolvido especialmente para famílias ou para uso próprio. O MyKeeper conta com funcionalidades de criação de listas de desejo - para aqueles momentos que você vai ao supermercado, mas sempre esquece de algo - e controle de estoque - podendo visualizar seus itens, separados por categoria e data de validade. Agora, nosso diferencial? O cadastro de produtos por comando de voz! Sim! O ato de cadastrar cada coisa que você comprou agora é fácil e rápido! Venha conhecer o MyKeeper!

## 📋 Funcionalidades

O sistema exige autenticação (login/senha) para acesso e oferece as seguintes seções:

* **🔐 Autenticação:** Sistema de login e cadastro para acesso seguro e personalizado ao seu estoque.
* **🗣️ Cadastro por Voz:** Adicione novos produtos ao estoque usando comandos de voz, de forma rápida e prática.
* **📦 Controle de Estoque:** Visualize e gerencie seus itens alimentícios, organizados por categoria e data de validade.
* **🛒 Lista de Desejos:** Crie listas de compras para não esquecer nenhum produto na próxima ida ao supermercado.
* **👤 Perfil:** Gerencie as informações da sua conta.

## ⚙️ Como rodar

Para rodar o projeto localmente, siga os passos abaixo:

### 1. Configuração do Banco de Dados
Antes de iniciar, é necessário configurar a conexão com o seu banco de dados local.

1. Abra o arquivo `conexao.php`.
2. Edite as variáveis com as credenciais do **SEU** ambiente local (host, usuário, senha).

### 2. Criação das Tabelas
Execute o script de criação para estruturar o banco de dados: `mykeeper.sql`

### 3. Execução do Projeto
Após configurar o banco:
* Crie sua própria conta na página de cadastro.
* Realize o login com a conta criada.
* Aproveite ao máximo do nosso sistema!!
