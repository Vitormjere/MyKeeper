# 💻 MyKeeper

> A food inventory management system built in PHP, designed to help you track your stock, manage shopping wishlists, and register products using voice commands.

The system allows users to register and manage their food items organized by category and expiration date, create shopping wishlists, and add products using voice commands, all stored in a MySQL database.

## 📋 Features

* **🔐 Authentication:** Secure login and registration system for personalized access to your inventory.
* **🗣️ Voice Registration:** Add new products to your stock using voice commands, quickly and easily.
* **📦 Stock Control:** View and manage your food items organized by category and expiration date.
* **🛒 Wishlist:** Create shopping lists so you never forget a product on your next trip to the supermarket.
* **👤 Profile:** Manage your account information.

## ⚙️ How to run

### 1. Database Setup

Create a MySQL database and run the SQL script to create the required tables:

```sql
mykeeper.sql
```

### 2. Configuration

Open the `conexao.php` file and fill in your local database credentials (host, user, password).

### 3. Running the project

* Set up the project in your XAMPP `htdocs` folder
* Start Apache and MySQL in the XAMPP Control Panel
* Create your account on the registration screen
* Log in and start managing your inventory!
